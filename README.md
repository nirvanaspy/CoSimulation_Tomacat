**Description：**
jsp+servlet+mysql实现的简单登录注册功能
----

#### 环境：
- ubuntu: 16.04
- mysql:  5.7.23
- jdk:    1.8
- idea:   2017
- tomcat: 9.0.10

#### 功能：
##### 注册
- 验证码
- 邮箱激活
- 用户名重复判断
- 邮箱重复绑定判断
- 密码md5加密存储

##### 登录
- 验证码
- session保存登录状态
